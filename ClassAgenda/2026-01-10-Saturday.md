## Today's Topics: Backup, Disaster Recovery, and Monitoring
> Focus: protecting workloads, planning for disasters, and monitoring Azure resources.

### Backup & Recovery
- **Azure Backup fundamentals**
  - What Azure Backup protects: VMs, files, databases, Azure Files
  - Recovery Services Vault: central management point
  - Backup policies: retention, frequency, backup types (full, incremental, differential)
  - Restore operations and recovery point objectives (RPO)
- **Azure Site Recovery (ASR)**
  - Disaster recovery scenarios: Azure-to-Azure, on-premises-to-Azure
  - Replication, failover, and failback concepts
  - Recovery plans and testing DR without impacting production
  - RPO vs RTO (Recovery Time Objective)

### Monitoring & Observability
- **Azure Monitor overview**
  - Metrics vs Logs: understanding the difference
  - Data sources: platform metrics, application insights, logs
  - Azure Monitor workspace organization
- **Log Analytics**
  - Kusto Query Language (KQL) basics
  - Common queries for troubleshooting
  - Creating alerts from log queries
- **Azure Monitor Alerts**
  - Alert types: metric alerts, log alerts, activity log alerts
  - Action groups: email, SMS, webhooks, runbooks
  - Alert processing rules
- **Application Insights** (brief overview)
  - Application performance monitoring (APM)
  - Dependency tracking and distributed tracing
- **Diagnostic Settings**
  - Routing logs and metrics to Log Analytics, Storage, Event Hubs
  - Retention policies and cost considerations

# Azure Automation - Coding Examples

This directory contains practical coding examples for Azure Automation, focused on preparing for the AZ-104 Azure Administrator certification.

## Learning Objectives

- Understand Azure Automation accounts and runbooks
- Create and manage automation runbooks using PowerShell and Python
- Work with Azure Automation variables, credentials, and schedules
- Implement update management for Azure VMs
- Use Azure Automation for resource deployment and configuration

## Examples Included

- `create-automation-account.ps1` - Create an Azure Automation account
- `create-automation-account.sh` - Create an Azure Automation account using Azure CLI
- `deploy-runbook.ps1` - Deploy and execute a PowerShell runbook
- `schedule-vm-start-stop.ps1` - Schedule VM start/stop automation
- `update-management-example.sh` - Configure update management for VMs

## Prerequisites

- Azure subscription
- Azure PowerShell module or Azure CLI installed
- Contributor access to resource group
- Basic understanding of PowerShell/Bash scripting

## AZ-104 Exam Topics Covered

- Deploy and manage Azure compute resources
- Monitor and maintain Azure resources
- Implement and manage storage
- Configure and manage virtual networks
